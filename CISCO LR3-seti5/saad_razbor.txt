4. Настройка ACL

- Пользователям из VLAN A должен быть разрешен доступ по любому протоколу к любым узлам сети, 
в т.ч. к сети Интернет.
- Пользователям из VLAN B должны иметь доступ к сети Интернет и к серверам PUBLIC и DNS 
по протоколам http и dns соответственно. Доступ к серверу LOCAL должен быть закрыт.
- Пользователям из VLAN C должен быть доступен только сервер LOCAL по протоколу http.
- Пользователям VLAN D должен быть доступен только Интернет, к серверам LOCAL и PUBLIC 
доступ должен быть закрыт.
- Между виртуальными сетями должен быть разрешен трафик протокола icmp.
- Из сети Интернет должен быть открыт доступ только по протоколам dns и http и только 
к серверам DNS и PUBLIC.

Сеть Интернет не имеет конкретного адреса. 
В моделируемой топологии сегмент Internet условно обозначает отдельный участок сети Интернет. 
В связи с этим адрес сети, в которой находятся internet user и internet server, не должен 
использоваться в конфигурации маршрутизаторов. 
Вместо этого при настройке ACL необходимо использовать ключевое слово any, а при настройке 
маршрутизации – маршруты по умолчанию.

Router B:

B>en
B#conf t
B(config)#ip access-list extended vlan14 
B(config-ext-nacl)#permit ip 10.2.13.0 0.0.0.15 any [vlan A, any]
B(config-ext-nacl)#permit icmp any any [any, any]

B(config-ext-nacl)#ip access-list extended vlan15
B(config-ext-nacl)#permit icmp any any [any, any]
B(config-ext-nacl)#deny ip 10.2.8.0 0.0.3.255 10.2.13.0 0.0.0.15 [vlan B, vlan A]
B(config-ext-nacl)#deny ip 10.2.8.0 0.0.3.255 10.2.12.0 0.0.0.127 [vlan B, vlan C]
B(config-ext-nacl)#deny ip 10.2.8.0 0.0.3.255 10.2.0.0 0.0.7.255 [vlan B, vlan D]
B(config-ext-nacl)#permit ip 10.2.8.0 0.0.3.255 any [vlan B, any]

B(config-ext-nacl)#ip access-list extended vlan16
B(config-ext-nacl)#permit icmp any any
B(config-ext-nacl)#permit ip 10.2.12.0 0.0.0.127 host 192.168.2.254 [vlan C, host 192.168.X.254]
B(config-ext-nacl)#permit udp 10.2.12.0 0.0.0.127 host 192.168.1.2 eq 53 [vlan C, dns]

B(config-ext-nacl)#ip access-list extended vlan17
B(config-ext-nacl)#permit icmp any any [any, any]
B(config-ext-nacl)#permit udp 10.2.0.0 0.0.7.255 host 192.168.1.2 eq 53 [vlan D, dns] 
B(config-ext-nacl)#deny ip 10.2.0.0 0.0.7.255 host 192.168.2.254 [vlan D, host 192.168.X.254]
B(config-ext-nacl)#deny ip 10.2.0.0 0.0.7.255 host 192.168.3.254 [vlan D, host 192.168.X+1.254]
B(config-ext-nacl)#permit ip 10.2.0.0 0.0.7.255 any [vlan D, any]
B(config-ext-nacl)#exit
B(config)#int f7/0.1
B(config-subif)#ip access-group vlan14 in
B(config-subif)#int f7/0.2
B(config-subif)#ip access-group vlan15 in
B(config-subif)#int f8/0.1
B(config-subif)#ip access-group vlan16 in
B(config-subif)#int f8/0.2
B(config-subif)#ip access-group vlan17 in

A(config-ext-nacl)#permit tcp any any established

A(config-ext-nacl)#permit tcp 10.2.8.0 0.0.3.255 host 192.168.3.254 eq 80 [vlan B, public]
A(config-ext-nacl)#permit udp 10.2.8.0 0.0.3.255 host 192.168.1.2 eq 53 [vlan B, dns]
A(config-ext-nacl)#deny ip 10.2.8.0 0.0.3.255 any [vlan B, any]
A(config-ext-nacl)#permit tcp 10.2.12.0 0.0.0.127 host 192.168.2.254 eq 80 [vlan C, 192.168.X.254]
A(config-ext-nacl)#permit tcp any host 192.168.3.254 eq 80 [any, 192.168.X+1.254]
A(config-ext-nacl)#permit udp any host 192.168.1.2 eq 53 [any, dns]
A(config-ext-nacl)#deny ip 10.2.12.0 0.0.0.127 any [vlan C, any]

A(config-ext-nacl)#permit tcp 10.2.0.0 0.0.7.255 host 192.168.2.254 eq 80 [vlan D, local]
A(config-ext-nacl)#deny ip 10.2.0.0 0.0.7.255 any [vlan D, any]
